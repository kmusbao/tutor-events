Tutor Events

SPA-приложение для управления учебными мероприятиями.

Приложение позволяет создавать, просматривать и управлять учебными событиями, отображать информацию об участниках и ответственных лицах.

Проект реализован на **React + TypeScript**.

---

## 🚀 Возможности

- ✅ Просмотр списка мероприятий
- ✅ Создание новых мероприятий
- ✅ Просмотр подробной информации о событии
- ✅ Отображение участников мероприятия
- ✅ Отображение ответственных лиц
- ✅ Валидация пользовательского ввода
- ✅ Material UI интерфейс

#
# 📦 Установка

## Клонирование проекта

```bash
git clone https://github.com/kmusbao/tutor-events.git

cd tutor-events
```

---

## Установка зависимостей

Используя Yarn:

```bash
yarn install
```

или npm:

```bash
npm install
```

---

# ▶️ Запуск проекта

Development режим:

```bash
yarn dev
```

или:

```bash
npm run dev
```

После запуска:

```text
http://localhost:5173
```


# 🧹 Проверка качества кода

Запуск ESLint:

```bash
yarn lint
```

Форматирование:

```bash
yarn prettier --write .
```
